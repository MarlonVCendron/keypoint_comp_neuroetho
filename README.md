#### Conda

##### Criar o ambiente

```
conda env create -f env.yml
```


##### Ativar o ambiente

```
conda activate keypoint_comp_neuroetho
```


##### Atualizar o ambiente

```
conda env update -f env.yml
```

##### Exportar o ambiente

```
conda env export --no-builds > env.yml
```
